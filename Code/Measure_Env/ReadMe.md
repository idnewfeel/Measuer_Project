# 环境监测

## 硬件 ##
* 主控：STM32F411RE
* 显示：LCD 240*240
* 温湿度传感器：SHT20(盛思瑞)
* TVOC传感器：SGP40(盛思瑞)只能测试空气质量指数，因为考虑SGP30的发热没有选择用SGP30
* 蜂鸣器
* 电池-充电管理电路
* 按键开机电路
* SD卡接口
* typc-C接口
* 功耗测量
----
## 软件
* LVGL
* 时间管理框架
